# [\#](https://super-apps.ai/mop/document/en/develop/frameInterface/fxs/annotation.html\#comments) Comments

FTS primarily has 3 methods for commenting.

#### [\#](https://super-apps.ai/mop/document/en/develop/frameInterface/fxs/annotation.html\#sample-code) Sample Code

```html
<!-- fxml -->
<fts module="sample">
// Method 1: Single-line comment

/*
Method 2: Multi-line comment
*/

/*
Method 3: End-of-file comment. All FTS code after /* is commented out

var a = 1;
var b = 2;
var c = "fake";

</fts>

        Code copied

```

In the above example, all FTS code is commented out.

> The only difference between Method 3 and Method 2 is that Method 3 does not have a \*/ terminator.
