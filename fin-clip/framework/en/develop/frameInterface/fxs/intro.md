# [\#](https://super-apps.ai/mop/document/en/develop/frameInterface/fxs/intro.html\#fts-introduction) FTS Introduction

FTS (Script) is a scripting language for Mini-Programs, which, combined with `FXML`, can be used to construct the structure of pages.

WXS and JavaScript are different languages; FTS has its own syntax and is not identical to JavaScript.

Please Note

FTS is compatible with WXS and WXML. You can use WXS and WXML for construction.
