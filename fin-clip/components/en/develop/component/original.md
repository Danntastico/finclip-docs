## [\#](https://super-apps.ai/mop/document/en/develop/component/original.html\#native-component-description) Native Component Description

Some components in Mini-Programs are native components created by the client. These components include:

- camera
- canvas
- input (only behaves as a native component when focused)
- live-player
- live-pusher
- map
- textarea
- video
