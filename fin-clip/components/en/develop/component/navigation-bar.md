## [\#](https://super-apps.ai/mop/document/en/develop/component/navigation-bar.html\#navigation-bar-2) navigation-bar

| Property | Type | Default Value | Required | Description | Minimum Version |
| --- | --- | --- | --- | --- | --- |
| title | string |  | No | Title of the navigation bar | 2.5.1 |
| loading | boolean | false | No | Whether to display a loading indicator in the navigation bar | 2.5.1 |
| front-color | string |  | No | Foreground color value of the navigation bar, including the color of buttons, title, and status bar; only supports #ffffff and #000000 | 2.5.1 |
| background-color | string |  | No | Background color value of the navigation bar; valid values are hexadecimal colors | 2.5.1 |
| color-animation-duration | number | 0 | No | Animation duration when changing the navigation bar color; default is 0 (no animation effect) | 2.5.1 |
| color-animation-timing-func | number | "linear" | No | Animation method when changing the navigation bar color; supports linear, easeIn, easeOut, and easeInOut | 2.5.1 |
