# [\#](https://super-apps.ai/mop/document/en/develop/api/forward.html\#forward-share) Forward / Share

## [\#](https://super-apps.ai/mop/document/en/develop/api/forward.html\#showsharemenu) showShareMenu

**showShareMenu(Object object)**

Displays the share button for the current page.

**Parameters**

**Object object**

| Property | Type | Default Value | Required | Description |
| --- | --- | --- | --- | --- |
| success | function |  | No | Callback function for successful API call |
| fail | function |  | No | Callback function for failed API call |
| complete | function |  | No | Callback function for when the API call ends (executed on both success and failure) |

## [\#](https://super-apps.ai/mop/document/en/develop/api/forward.html\#hidesharemenu) hideShareMenu

**hideShareMenu(Object object)**

Hides the share button for the current page.

**Parameters**

**Object object**

| Property | Type | Default Value | Required | Description |
| --- | --- | --- | --- | --- |
| success | function |  | No | Callback function for successful API call |
| fail | function |  | No | Callback function for failed API call |
| complete | function |  | No | Callback function for when the API call ends (executed on both success and failure) |
