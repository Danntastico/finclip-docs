# [\#](https://super-apps.ai/mop/document/en/introduce/update/fino-update-log.html\#finclip-app-update-log) FinClip App Update Log

Get the Latest Version

To ensure the best experience, please update the FinClip App on your phone to the latest version;

- iPhone users, please log in to the App Store and search for " [FinClip - FinClip App(opens new window)](https://apps.apple.com/cn/app/finclip-%E5%87%A1%E6%B3%B0%E5%8A%A9%E6%89%8B/id1530202233)" to download;
- Android users, please click the [Download Link(opens new window)](https://mp.finogeeks.com/mop/scattered-page/#/mop-download), or go to "Tencent MyApp" or "Google Play Store" to download the latest version of the FinClip App;

Update Notice

- The FinClip App is primarily used to update the latest SDK information, so we may not maintain the App's version update notes in real-time;
- You can view the latest information for each SDK version here ( [iOS](https://super-apps.ai/mop/document/runtime-sdk/ios/ios-update-log.html), [Android](https://super-apps.ai/mop/document/runtime-sdk/android/android-update-log.html)), or click the link above to download the corresponding latest version.
